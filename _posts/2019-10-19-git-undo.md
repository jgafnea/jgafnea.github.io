# Undo commits

```sh
# Undo last commit and unstage
$ git reset HEAD~

# Undo last commit but don't unstage
$ git reset --soft HEAD^
```

# Undo commit message

```sh
# Open editor
$ git commit --ammend

# Inline
$ git commit --ammend -m "New commit message"
```

# Undo checkout

```sh
$ git checkout -
```
